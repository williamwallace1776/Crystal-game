Welcome to the crystal game!

  • At the start of the game you will be given a random number
  • The four crystals have a number that corresponds to them.
  • Clicking on said crystals will make your score increase by a specific amount.
  • The path to victory is making your score match the number given without going over
  • But there is a catch, the value of each crystal is hidden from view
  • At the start of each game, both the Target Score and the value of each individual crystal will change
